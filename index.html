Kebijakan Privasi - Do'a Islami

Terakhir diperbarui: [Tanggal]

1. Pengenalan

Aplikasi Do'a Islami ("aplikasi kami") berkomitmen untuk melindungi privasi pengguna. Kebijakan privasi ini menjelaskan bagaimana kami menangani informasi Anda.

2. Informasi yang Kami Kumpulkan

Do'a Islami TIDAK mengumpulkan data pribadi apa pun:

  • ❌ Tidak mengumpulkan nama, email, atau informasi kontak
  • ❌ Tidak mengumpulkan data lokasi GPS
  • ❌ Tidak mengakses kontak atau pesan
  • ❌ Tidak menggunakan kamera atau mikrofon
  • ❌ Tidak melacak aktivitas browsing

3. Data yang Disimpan Secara Lokal

Aplikasi hanya menyimpan preferensi lokal di perangkat Anda:

  • ✅ Posisi terakhir membaca do'a
  • ✅ Preferensi tema (jika ada)
  • ✅ Pengaturan font size (jika ada)

Semua data ini disimpan secara lokal dan tidak dikirim ke server mana pun.

4. Izin yang Digunakan

Aplikasi meminta izin berikut:

  • INTERNET - Untuk update konten do'a di masa depan (opsional)
  • ACCESS_NETWORK_STATE - Untuk memeriksa koneksi internet

Izin ini hanya digunakan untuk fungsi yang dimaksud dan tidak untuk tracking.

5. Layanan Pihak Ketiga

Do'a Islami tidak menggunakan:

  • ❌ Tidak ada analytics (Google Analytics, Firebase, dll.)
  • ❌ Tidak ada iklan (Google AdMob, dll.)
  • ❌ Tidak ada social media integration
  • ❌ Tidak ada third-party SDK

6. Kebijakan untuk Anak-Anak

Do'a Islami:

  • ✅ Cocok untuk semua usia
  • ✅ Tidak mengumpulkan data dari anak di bawah 13 tahun
  • ✅ Konten aman dan edukatif
  • ✅ Tidak ada fitur sosial atau chat

Jika kami mengubah kebijakan privasi ini, kami akan memperbarui tanggal "Terakhir diperbarui" dan memposting versi baru di aplikasi dan website.
